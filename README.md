# 🛒 Product Billing System (Java)

A simple **Console-Based Product Billing System** developed using **Core Java**. This mini project allows users to add products, view available products, purchase items, and generate a bill. It demonstrates the use of Object-Oriented Programming (OOP) concepts and Java Collections.

---

## 📌 Features

- ➕ Add new products
- 📋 View available products
- 🛍️ Purchase products
- 📦 Automatic stock update after purchase
- 🧾 Generate and display bill
- 💰 Calculate Grand Total
- 🚫 Prevent purchasing more than available stock
- ❌ Handle invalid product selection

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class
- VS Code / Eclipse / IntelliJ IDEA

---

## 📂 Project Structure

```
Project
│
├── Product.java      // Stores product details
├── BillItem.java     // Stores purchased product details
└── Main.java         // Main program with menu-driven operations
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Product-Billing-System.git
```

2. Open the project in your preferred Java IDE.

3. Compile the program

```bash
javac Main.java
```

4. Run the program

```bash
java Main
```
